# 组合数

```cpp
fac[1] = 1;
for(int i = 2;i <= 5000;i++)
    fac[i] = fac[i - 1] * i % p;
inv[5000] = ksm(fac[5000],p - 2); // 求 fac[5000] 的 逆元
for(inti = 5000;i >= 2;i--)
    inv[i - 1] = inv[i] * i % p;  // 1 / n! * n = 1 / (n - 1)!
```    

$C^a_b$ = `fac[b] * inv[a] * inv[b - a]`
